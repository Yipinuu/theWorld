添加 Slack_notification 当试用 Capistrano 部署时

这个项目之前就已经使用了 capistrano 做部署，现在是为了 Slack 上能够收到部署的通知。

我们首先要在 Slack 上开通接受通知的服务，这个服务叫做 incoming webhooks。

开通后，注意 Slack 给你的 Webhook URL。

首先在 deploy/production.rb 添加这个 Channel name 和 Webhook Url。

然后是一个可以通用的 deploy/recipes/slack_notification.rb

我们要告诉 Cap 注意 recipes 这个文件夹

做完这一切的时候呢，如果你的 capistrano 版本是 3.* 。当你使用 cap 部署时，会有报错。

点击这个 参照它的意见去做。
