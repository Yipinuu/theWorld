Go语言编程入门（九）：函数

每一个程序都包含很多函数
函数是基本的代码块

  • Go是编译型语言，所以函数的编写顺序是无关紧要的
  • 鉴于可读性，将 main() 函数写在文件的前面，其他函数按照一定逻辑顺序
  • 函数会执行到代码块最后一行（}之前）或者 return 语句时推出。

Go 里面拥有三种类型的函数

  • 普通的带有三种类型的函数
  • 匿名函数或者lambda函数
  • 方法

greeting.go

如果需要神明一个在外部定义的函数，你只需要给出函数名与函数签名，不需要给出函数体

函数也可以以申明的方式被使用，作为一个函数类型

函数参数与返回值

  • 函数能接受参数供自己使用，也可以返回零或者多个值
  • 任何一个有返回值的函数都必须以 return 或 panic 结尾
  • 没有参数的函数，通常被称为 niladic 函数
  • Go 传递的是参数的副本，所以在函数中使用变量的过程，对副本的值进行更改，不会影响到原来的变量
  • 如果你想要直接影响到参数，你需要将参数的指针传递给函数，我们可以通过修改指针的值来修改这个值所指向的地上的值。
  • 几乎在任何情况下，传递指针（一个32位或者64位的值）的消耗都比传递副本来的少。
  • 在函数调用时，像切片（slice）、字典（map）、接口（interface）、通道（channel）这样的引用类型都是默认使用引用传递（即时没有显示的指出指针）
  • 如果一个函数需要返回四到五个值，我们可以传递一个切片给函数（如果返回值具有相同类型）或者是传递一个结构体（如果返回值具有不同的类型）
  • 尽量使用命名返回值：会使代码更清晰、更简短，同时更加容易读懂代码 multiple_return.go
  • 空白符用来匹配一些不需要的值，然后丢弃。blank_identifier.go
  • 通过指针我们可以修改外部变量side_effect.go
