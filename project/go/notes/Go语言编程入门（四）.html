Go语言编程入门（四）：基本类型与运算符与字符串

数字类型

  • int8 (-128 -> 127)
  • int16 (-32768 -> 32767)
  • int32 (-2,147,483,648 -> 2,147,483,647)
  • int64 (-9,223,372,036,854,775,808 -> 9,223,372,036,854,775,807)
  • uint8 (0 -> 255)
  • uint16 (0 -> 65,535)
  • uint32 (0 -> 4,294,967,295)
  • uint64 (0 -> 18,446,744,073,709,551,615)
  • float32 (+-1e-45 -> -3.4*1e38)
  • float64 (+-51e324 -> 1071e308)
  • complex64 (32位实数和虚数)
  • complex128 (64位实数和虚数)

你可以通过增加前缀 0 来表示 8 进制数（如： 077），增加前缀 0x 来表示 16进制数 （如： 0xFF），以及使用 e 来表示10的连乘（如： 1e3 = 1000, 或者 6.022e23 = 6.022*1e23）。

\x总是紧跟着长度为 2 的 16进制数，\后面跟着长度为3的十进制数，\u 后面紧跟着长度为4的16进制的 Unicode 字符，\U 后面总是紧跟着长度为8的16进制数。

Go 中不允许不同类型之间的混合使用，但是对于常量的类型限制非常少，因此允许常量之间的混合使用。

格式化说明符

  • %d 用于格式化整数
  • %x(%X) 用于格式化16进制表示的数字
  • %g 用于格式化浮点型
  • %f 用于输出浮点数
  • %e 输出科学计数表示法
  • %0d 用于规定输出定长的证书，其中开头的数字必是0
  • %v
  • %c 用于表示字符
  • %U 输出格式为U+hhhh的字符串

复数

使用 re+imI 来表示，其中 re 代表实数部分， im 代表虚数部分， I 代表根号负 1。

位运算

二元运算符 按位与& 按位或｜ 按位异或^ 位清除&^
一元运算符 按位补足^ 位左移<< 位右移>>

unicode 字符

  • 判断是否为字母： unicode.IsLetter(ch)
  • 判断是否为数字： unicode.IsDigit(ch)
  • 判断是否为空白符号： unicode.IsSpace(ch)

字符串

字符串是utf-8的一个序列（当字符为ASCⅡ码时则占用1个字节，其他字符需要占用2-4个字节）。
字符串是一种值类型，且值不可变，即创建某个文本后你无法再次修改这个文本的内容；更深入地讲，字符串是字节的定长数组。
