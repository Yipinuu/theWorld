Go语言编程入门（三）：常量与变量

常量

  • 常量使用关键字 const 定义。 储存在常量的数据类型，只能是布尔型，数字型和字符串型。

  • 常量的定义是格式

  • Go语言中常量的定义可以省略说明符[type]

  • 常量的值必须在编译中就可以确定的。如：它不能是自定义函数，自定义函数属于未知。但是可以使用内置函数。
  • 常量允许并行赋值，也可以用作枚举。
  • 常量的大小是可以由精度来定义的，低精度赋值给高精度容易溢出。
  • iota 初始值为0，每在新一行被使用时，值加1。

变量

  • 变量使用关键字 var 定义。

  • 变量的定义格式。变量将变量类型放在变量名称之后。

  • 当变量被申明后，系统会自动赋予它们该类型的零值。

  • 变量名的书写采用驼峰式。但如果你期待你的变量被外部包所使用，则首字母需要为大写。
  • 变量可以在编译时被赋值，也可以在运行时被赋值。

  • 当你在函数体内定义局部变量，应该使用 :=。

  • 打印
    = Printf 主要用户打印到控制台，通常使用格式化字符串作为第一个参数
    = Print 自动使用格式化标识符%v对字符串进行格式化，两者都会在每个参数之间自动增加空格，
    = Println 基本与Print相同，但会在字符串的最后加上一个换行符

  • 空白标识符 _ 也被用于作为抛弃值
  • init函数。每个源文件可以有且只有一个init函数。初始化总是以单线程执行，并且按照包的依赖关系顺序执行。
